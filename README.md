# Shopping-Cart-System

## Overview
* Developed a shopping app that allows users to search different products available
* It is a web application built using JAVA and REACT
* Used JWT to Authenticate User credentials.
* Online Database MongoDB to store and fetch Data.
* Users can Pay for order using Paytm.
* Complete the Admin interface to check all order details and edit User’s data also admins can add new products and edit details as needed.
* Technology Used - JAVA, Postman, REACT, Eureka Server, Cloud as an API Gateway, Swagger.
